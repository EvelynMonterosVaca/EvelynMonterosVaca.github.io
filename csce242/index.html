Assignment 8 - Arrays

For this assignment, we get to follow a fun outline while having the freedom to pick our own color scheme! Your task is to create an associative array that includes the titles and the six images we've been given. When the page loads, you’ll loop through the array to show all the titles. When you click on a title, a pop-up box will appear displaying the title, the matching image, and a close button for easy access. You'll be pulling this info straight from the associative array. Don't forget to add that close button in the pop-up! And remember, we need to give credit for the images we use. Have fun with it!

Assignment 9 - Classes

Create a Pizza class that includes details like the pizza's image filename and a method to return an HTML section for its title and image. Implement a popup that shows class variable data when the pizza's name or image is clicked.

Create an array of at least five pizzas with relevant details and store their images in a dedicated folder. Loop through this array to dynamically add each pizza to the DOM, ensuring a visually appealing layout with uniform image widths. If you have trouble with the popup functionality, you can display the information directly on the page as an alternative.
